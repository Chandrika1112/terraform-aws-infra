# Terraform AWS Infrastructure Automation

This project demonstrates **Infrastructure as Code (IaC)** using **Terraform** to provision and manage AWS cloud resources in an automated, repeatable, and scalable way.

The goal of this project is to eliminate manual AWS resource creation and enable consistent infrastructure deployment using Terraform configuration files.
